# 车牌号码样式优化说明

## 修改目的

在车辆预约详情弹窗中，为车牌号码添加视觉样式，使其更具辨识度和美观性，模拟真实车牌外观。

## 修改内容

### 文件
`d:/nefu-edu-datav/vue-big-screen-master/src/views/VisitorPlanDashboard.vue`

### 1. HTML 结构修改（第71-73行）

**修改前**:
```vue
<td v-if="modalType === 'vehicle'">{{ item.carNumber }}</td>
```

**修改后**:
```vue
<td v-if="modalType === 'vehicle'">
  <span class="car-plate">{{ item.carNumber }}</span>
</td>
```

### 2. CSS 样式添加（第990-1019行）

添加 `.car-plate` 样式类：

```scss
.car-plate {
  display: inline-block;
  padding: 6px 12px;
  background: linear-gradient(135deg, #1e90ff 0%, #0066cc 100%);
  color: #ffffff;
  font-weight: bold;
  font-size: 14px;
  font-family: 'Arial Black', 'Microsoft YaHei', sans-serif;
  letter-spacing: 2px;
  border-radius: 4px;
  border: 2px solid rgba(255, 255, 255, 0.3);

      rgba(255, 255, 255, 0.15), 
      transparent);

  }
}
```

## 样式特点

### 1. 蓝色渐变背景
```scss
background: linear-gradient(135deg, #1e90ff 0%, #0066cc 100%);
```
- 从浅蓝色 `#1e90ff` 到深蓝色 `#0066cc` 的对角线渐变
- 模拟中国蓝色车牌的颜色

### 2. 白色加粗字体
```scss
color: #ffffff;
font-weight: bold;
font-size: 14px;
font-family: 'Arial Black', 'Microsoft YaHei', sans-serif;
letter-spacing: 2px;
```
- 白色文字，高对比度
- 加粗字体，更醒目
- 字符间距 2px，字符分散，更易识别

### 3. 边框和圆角
```scss
border-radius: 4px;
border: 2px solid rgba(255, 255, 255, 0.3);
```
- 圆角 4px，柔和边缘
- 半透明白色边框，增加层次感

### 4. 阴影效果
```scss
box-shadow: 
  0 2px 8px rgba(30, 144, 255, 0.4),
  inset 0 1px 0 rgba(255, 255, 255, 0.2);
text-shadow: 0 1px 2px rgba(0, 0, 0, 0.3);
```
- 外阴影：蓝色光晕，增加立体感
- 内阴影：顶部白色高光，模拟反光效果
- 文字阴影：黑色阴影，增强文字立体感

### 5. 高光效果
```scss
&::before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  height: 40%;
  background: linear-gradient(to bottom, 
    rgba(255, 255, 255, 0.15), 
    transparent);
  border-radius: 2px 2px 0 0;
}
```
- 使用伪元素创建顶部高光
- 渐变从半透明白色到透明
- 覆盖车牌上半部分，模拟光线反射

## 样式参数调整建议

如需调整样式，可以修改以下参数：

### 1. 调整车牌背景色
```scss
// 蓝色车牌（默认）
background: linear-gradient(135deg, #1e90ff 0%, #0066cc 100%);

// 绿色新能源车牌
background: linear-gradient(135deg, #00c853 0%, #00a846 100%);

// 黄色车牌
background: linear-gradient(135deg, #ffc107 0%, #ff9800 100%);
color: #000000; // 黄色车牌使用黑色字体
```

### 2. 调整字体大小
```scss
font-size: 14px; // 默认
font-size: 16px; // 更大
font-size: 12px; // 更小
```

### 3. 调整字符间距
```scss
letter-spacing: 2px; // 默认
letter-spacing: 3px; // 更宽
letter-spacing: 1px; // 更紧凑
```

### 4. 调整内边距
```scss
padding: 6px 12px; // 默认
padding: 8px 16px; // 更大
padding: 4px 10px; // 更小
```

## 可扩展性

如果未来需要支持不同类型的车牌，可以添加动态类名：

```vue
<span :class="['car-plate', getPlateType(item.carNumber)]">
  {{ item.carNumber }}
</span>
```

```javascript
methods: {
  getPlateType(carNumber) {
    if (carNumber && carNumber.length > 7) {
      return 'green-plate'; // 新能源车牌
    }
    return 'blue-plate'; // 普通蓝牌
  }
}
```

```scss
.car-plate {
  &.blue-plate {
    background: linear-gradient(135deg, #1e90ff 0%, #0066cc 100%);
  }
  
  &.green-plate {
    background: linear-gradient(135deg, #00c853 0%, #00a846 100%);
  }
  
  &.yellow-plate {
    background: linear-gradient(135deg, #ffc107 0%, #ff9800 100%);
    color: #000000;
  }
}
```

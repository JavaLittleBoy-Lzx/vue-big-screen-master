# 🚗 智慧停车3D数据分析大屏

## 📋 项目概述

基于Vue.js和ECharts开发的智慧停车场3D数据分析大屏，将停车时长分析和通道进出情况分析合并展示，采用3D立体热力图等科技感图表组件，提供直观的数据可视化体验。

## 🎯 核心功能

### 1. 3D立体热力图
- **时空三维分析**：时间(24小时) × 停车时长(4个区间) × 车辆数量
- **实时数据更新**：动态展示停车场使用情况
- **交互式控制**：时间轴滑动查看不同时段数据
- **热力分布**：颜色渐变直观显示车流量密度

### 2. 飞线流动图
- **通道流动轨迹**：动态展示车辆在通道间的流动
- **实时连接线**：显示入口、出口、中央区的连接关系
- **流动模式分析**：识别车辆流动规律和瓶颈

### 3. 实时KPI监控
- **总车流量**：实时统计进出车辆总数
- **平均停车时长**：计算加权平均停车时间
- **通道效率**：评估通道使用效率
- **利用率峰值**：监控停车场饱和状态

## 🛠️ 技术栈

- **前端框架**：Vue.js 2.x
- **图表库**：ECharts 5.x
- **样式预处理**：SCSS
- **构建工具**：Vue CLI
- **路由管理**：Vue Router

## 📊 数据维度分析

### 时间维度
- **24小时时段分析**：全天候监控停车场使用情况
- **高峰时段识别**：自动识别车流量高峰时段
- **流量变化趋势**：分析不同时段的流量变化规律

### 时长维度
- **短停(0-1h)**：快速停车，适合临时停车需求
- **中停(1-4h)**：常规停车，主要停车类型
- **长停(4-8h)**：长时间停车，工作日停车特征
- **超长停(>8h)**：异常停车，需要关注和预警

### 通道维度
- **入口通道效率**：分析各入口的处理能力
- **出口通道效率**：评估出口疏散效率
- **通道间流动模式**：识别车辆流动规律
- **瓶颈识别**：发现通道使用瓶颈

### 流动维度
- **车辆流动轨迹**：可视化车辆移动路径
- **通道转移模式**：分析车辆在通道间的转移
- **流量平衡分析**：评估进出流量平衡状况
- **拥堵预警**：预测和预警拥堵情况

## 📈 数据接口

### 热力图数据格式
```javascript
// 数据格式：[小时, 时长索引, 车辆数量]
const heatmapData = [
  [0, 0, 15],   // 0点，短停，15辆车
  [0, 1, 8],    // 0点，中停，8辆车
  [1, 0, 12],   // 1点，短停，12辆车
  // ...
]
```

### 飞线图数据格式
```javascript
const flylineData = {
  points: [
    { coordinate: [0.2, 0.3], text: '入口1' },
    { coordinate: [0.8, 0.3], text: '入口2' },

  lines: [
    { source: '入口1', target: '中央区' },
    { source: '中央区', target: '出口1' },
    // ...
  ]
}
```
